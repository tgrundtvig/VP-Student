Welcome to the Advanced Programming course! This is a 14-week evening course designed for adult learners, focusing on modern software design principles and professional development practices.

## Course Philosophy

This course teaches **interface-first, top-down design thinking** - a powerful methodology for building maintainable, testable, and professional software systems. You'll learn to design first, then implement, following the principle of "wishful programming."

## Course Structure

### Format
- **Duration**: 14 weeks + 2-week implementation period
- **Schedule**: One evening per week, 17:00-20:00 (3 hours)
- **Language**: English (all written materials and code)

### Course Phases

**Phase 1: Discovery (Weeks 1-4)**
Experience the pain of poor design, then discover interface-first solutions

**Phase 2: Mastery (Weeks 5-10)**
Deep dive into data structures, algorithms, and design patterns through interfaces

**Phase 3: Application (Weeks 11-14)**
Design your exam project with peer feedback and refinement

**Phase 4: Implementation (2-week gap)**
Implement your designed system independently

## Learning Outcomes

By the end of this course, you will be able to:

- Design software systems using interface-first methodology
- Apply top-down thinking to break complex problems into manageable layers
- Distinguish between data (records) and behavior (interfaces)
- Write testable code with clear separation of concerns
- Implement data structures and algorithms from first principles
- Apply design patterns naturally through interface-driven design
- Build complete applications using modern Java practices

## Important Principles

### Interface-First Design
1. Define what modules do (interfaces) before how they do it (implementation)
2. Use records for data, interfaces for behavior
3. Design all layers before implementing the leaves
4. Write tests against interfaces, not implementations

### Code Standards
- **All code in English** - variables, methods, classes, comments
- **Clear naming** - code should read like prose
- **Comprehensive tests** - prove your design works
- **Professional practices** - this prepares you for real work
