## For The Exam

See [exam-guide.md](../exam-guide.md) for the full preparation guide.

Quick checklist for the week leading up to the oral exam:
- [ ] Can I draw my project's top-level interfaces from memory?
- [ ] Can I justify every design decision in 1 sentence?
- [ ] Can I name the design patterns I used and where?
- [ ] Do I have tests against my interfaces (contract tests)?
- [ ] Can I demo the project running, end to end?
- [ ] Can I answer "why an interface and not a class here?" for any class
      in my project?
