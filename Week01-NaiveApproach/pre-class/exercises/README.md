# Pre-Class Exercises: Java Fundamentals

These exercises verify your understanding of basic Java concepts needed for Week 1.

## Setup

1. Ensure you have Java 21 and Maven installed (see [setup-guide.md](../../../setup-guide.md))
2. Navigate to this directory in your terminal
3. Run `mvn clean test` to see which tests fail

## Exercises

Complete the TODO sections in these files:

1. **Character.java** - Basic class with fields and methods
2. **Inventory.java** - Working with collections
3. **CombatCalculator.java** - Simple game logic

## Running Tests

```bash
# Run all tests
mvn test

# Run tests for specific class
mvn test -Dtest=CharacterTest

# Compile without testing
mvn compile
```
