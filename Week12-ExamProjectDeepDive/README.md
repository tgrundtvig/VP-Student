# Week 12: Exam Project Deep Dive

**Date:** 5 May 2026

## What We Did In Class

Second exam-project guidance session. Continued one-on-one work on each
student's project, now focused on:
- **Reviewing the interface design** from Week 11
- **Identifying gaps** by trying to write workflow code against the interfaces
- **Refining records vs interfaces** decisions
- **Picking implementation order** — which leaf node to implement first?

Same iterative loop as Week 11 — design first, then try to use what you
designed, let the gaps in your code tell you what's missing.

## For The Exam

By end of this session you should have:
- A complete top-level interface design (no implementation needed yet).
- Workflow code that uses *only* your interfaces — proves the design works
  on paper before you commit to building it.
- A prioritised list of which implementations to build first.

The implementation phase runs between session 14 (19 May) and hand-in
(**27 May 2026**). The oral exam is **9 June 2026**.
